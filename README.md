# Клеточный футбол (Lab_02_cell-football)

Учебный проект по дисциплине: ЖЦРПО (Жизненный цикл разработки ПО) **Лабораторная работа №2** — разработка и документирование требований к программному обеспечению.

**Автор:** Гузаревич Роман Алексеевич  
**Группа:** 250541  

---

## О проекте

Мобильное приложение — цифровая версия игры в стиле «футбол на бумаге»: клетчатое поле, пошаговые ходы, ворота, правила **v1.0** зафиксированы в отдельном документе.

**Версия 1.0 (по SRS):** игра против **ИИ**, **локальный PvP** на одном устройстве, **ручное ведение турниров** на стороне клиента. **Удалённый онлайн-PvP в v1.0 не входит.**

**Phase 2 (план):** онлайн-PvP между удалёнными игроками с использованием **сетевого компонента** (облачный сервис или собственный API).
